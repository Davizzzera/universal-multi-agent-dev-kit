# Landing Page Pack

> A preset for marketing pages, conversion pages, product pages, lead capture pages and campaign pages.

## Pack Overview
This pack curates the optimal configuration of agents, skills, and workflows to build high-conversion, accessible, and fast-loading landing pages with strong copy and SEO.

## When to use this pack
- Landing pages
- Product presentation pages
- Lead generation
- Conversion optimization
- Campaign pages
- Public website sections

## When NOT to use this pack
- Complex backend systems
- Database-heavy applications
- Internal automation-only projects

## Recommended Agents
This pack utilizes marketing strategists, copywriters, and frontend specialists. See `agents.md` for the full list.

## Recommended Skills
This pack relies on copywriting, SEO, UX flows, and layout audits. See `skills.md` for the full list.

## Recommended Workflows
This pack uses workflows that enhance UI and copy, such as `frontend-ui` and `enhance-feature`. See `workflows.md` for the full list.

## Validation Expectations
Visual consistency, responsiveness across devices, and accessibility audits must pass. SEO metrics (title, description, semantic HTML) should be verified.

## Safety and Scope Notes
- Do not modify backend services or APIs with this pack.
- Ensure the `scope-guardian` restricts edits to public assets and frontend components.

## Example Use Cases
- "Create a new Black Friday promotional landing page."
- "Optimize the existing pricing page for better conversion."
